    Important Python Modules

    Glob: - Function name glob()
    • overview : The glob function in the glob module accepts file patterns and returns a list of file path that meets the pattern
    • i/p : File Pattern
    • o/p : List of file paths
    CSV : - Function name reader()
    • overview : The reader function in csv accepts a file object and returns an iterator with File Content
    • i/p : File Object
    • o/p : Iterator with file content
    Shutil : Function name make_archive()
    • overview : make_archive function accepts the name, type and directory; and creates an Archieve File
    • i/p : Name type and Filepath/directory
    • o/p : No output but creates an archived file/zip File
    Webbroweser : Functiona name open()
    • overview : open function accepts a url and opens the mentioned url with the default browser
    • i/p : URL
    • o/p : opens the URL

    Modules

    • Module: You can store all defined functions in python modules and then access any of them by import
    • You can import in two ways
      • “Import FileName” - You can call the function with FileName.Function() 
      • “from FileName import function” - You can call the function directly 
    • When you import the python file will be executed
      • During the import __name__ will have the value of name of the file
      • If you execute the python module directly __name__ will have the value “__main__”
    • If you have Code outside the functions, it’ll be executed when you import

    Default Parameter

    • Default Parameter - When you don’t want to pass the argument every time you call the function you can define a default value for the parameter
    • Parameters with default values should be defined after the parameter without default values
    • When you call the function that has a default value parameter defined you don’t need to pass the value again
    • You can also pass a value even if the default value is defined.
    • Inbuilt python functions can also have default variables

    Any & All

    • To check if any item or all items in an iterable evaluate to True, you can use any() and all() built in functions
    • any() returns true when any one item is true
    • all() returns true when all items are true

    Iterable

    • is an object which can be looped over or iterated over with the help of a for loop
    • Objects like lists, tuples, sets, dictionaries, strings, etc. are called iterables.

    What strings, numbers & iterable are true ?

    • All strings are True except for empty ones
    • All> numbers are true except0
    • All lists, sets , tuples and dictionaries are true except for empty ones
