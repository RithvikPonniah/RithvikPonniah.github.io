    Default Parameter

    • Default Parameter - When you don’t want to pass the argument every time you call the function you can define a default value for the parameter
    • Parameters with default values should be defined after the parameter without default values
    • When you call the function that has a default value parameter defined you don’t need to pass the value again
    • You can also pass a value even if the default value is defined.
    • Inbuilt python functions can also have default variables

    Any & All

    • To check if any item or all items in an iterable evaluate to True, you can use any() and all() built in functions
    • any() returns true when any one item is true
    • all() returns true when all items are true

    Iterable

    • is an object which can be looped over or iterated over with the help of a for loop
    • Objects like lists, tuples, sets, dictionaries, strings, etc. are called iterables.
